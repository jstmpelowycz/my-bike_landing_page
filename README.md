## Description
### General information:
1. Here is a responsive landing page

2. Features:
    - interactive links and buttons
    - hover effects applied to every card in sections **Compare Bikes** and **Details**

### Tech stack:
- HTML5
- CSS3
- Sass (SCSS)
- BEM methodology
- Flexbox/Grid
- JavaScript (form validation, animations for button-links, menu dropdown)

## Local development
### Dependencies
* Node v14.18.2
* NPM v6.14.15

### How to install
* Fork and clone repo
* Run `npm install`
* Run `npm start`
